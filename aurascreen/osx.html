MacOS Installation Instructions

Information

Since Aura Screen is developed in C# WinForms .Net, it only works natively in Windows. This is why we need your support with donations, so we can hire a competant Swift/OBJ-C Developer. Until a native MacOS port is made, there are a few options to run the Windows version, some cost money, some are free. Choose the method you are most comfortable with.

Method 1: Parallels

MacOS Parallels - The most robust, but costs $$ Using Parallels gives you some added functionality via the Comprehensive mode which will overlay a Windows Program ontop of your MacOSX desktop. Parallels, does cost money. If you already have it great! Setup is easy!

Download Parallells

Follow the Parallels setup instructions. Once Parallels is installed and setup, simply follow the same instructions for the Windows Section!

Setup Instructions

Method 2: MacOS Bootcamp

MacOS Bootcamp - Identical to the Windows Use case as you just boot windows. Technically free Follow the Bootcamp setup from Apple Boot into the new Windows Partition, and then follow the Windows instructions You can migrate from Bootcamp to Paralles in the future for ease of use without having to reinstall Windows!

Method 3: MacOS VMWare

Using VMWare works exactly like Parallels, but does not have the comprehensive mode. Allowing for a bit less customization. Install your Windows VM following the instructions from VMWare VMWare Setup Aura Screen in the Guest OS as you would normally in Windows.
